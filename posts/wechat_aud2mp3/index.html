Wechat_aud2mp3

Come a request that merge the audios in the wechat to a file of mp3. The following is the steps, after more than a week’s study…

  1. get the raw data files via itunes backup. Note that here is based on iphone setting.

    • backup by itunes without encrypted, removing profile in general setting.
    • use iExpoler app to get the backup files. There is an “Audio” folder which contains the aud files.
    • find . -name "*.aud" -newermt 2019-06-13 ! -newermt 2019-06-14 | xargs -I{} cp {} 0613

  2. transfer the aud files to mp3 files

    • ./converter.sh ../../0622/4th/aud/ ../../0622/4th/mp3/ mp3

  3. merge to a whole mp3 file

    • make a list: ls *.mp3 | xargs -t -I{} echo file \'{}\' >> mylist.txt
    • concate to a file: ffmpeg -f concat -i mylist.txt -c copy output.mp3
